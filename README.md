# 🤖 JARVIS Voice Assistant

A Python-based voice assistant with speech recognition, AI integration, and GUI.

## Features
- Wake word activation ("Jarvis")
- Voice commands
- AI responses (OpenAI)
- Weather API integration
- Desktop GUI

## Technologies
- Python
- SpeechRecognition
- pyttsx3
- OpenAI API
- CustomTkinter

pip install -r requirements.txt
