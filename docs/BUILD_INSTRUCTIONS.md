# 📦 Сборка EXE файла голосового ассистента

## 🎯 Что это?

Скрипты для создания **автономных .exe файлов**, которые работают **без установки Python**.

---

## 🚀 Быстрая сборка

### Способ 1: Автоматический (рекомендуется)

```bash
# Просто запустите:
build.bat
```

Скрипт автоматически:
1. Создаст виртуальное окружение
2. Установит все зависимости
3. Соберёт **JarvisCLI.exe** (консольная версия)
4. Соберёт **JarvisGUI.exe** (графическая версия)

### Способ 2: Ручная сборка

```bash
# 1. Установка PyInstaller
.venv\Scripts\python.exe -m pip install pyinstaller

# 2. Сборка CLI версии
pyinstaller JarvisCLI.spec

# 3. Сборка GUI версии
pyinstaller JarvisGUI.spec
```

---

## 📂 Результат сборки

После успешной сборки в папке `dist/` появятся:

```
dist/
├── JarvisCLI.exe     ← Консольная версия (голосовое управление)
└── JarvisGUI.exe     ← Графическая версия (окно с кнопками)
```

## 💡 Как использовать .exe файлы

### 1. Копирование
Скопируйте `.exe` файл в любую папку, например:
```
C:\Program Files\Jarvis\
```

### 2. Создание конфигурации
Создайте файл `.env` рядом с `.exe`:

```env
# Конфигурация голосового ассистента
GROQ_API_KEY=gsk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
AI_PROVIDER=groq
TTS_PROVIDER=edge
MICROPHONE_INDEX=
```

**Получить бесплатный API ключ:** https://console.groq.com/keys

### 3. Запуск
- **JarvisCLI.exe** — запустите двойным кликом (консольная версия)
- **JarvisGUI.exe** — запустите двойным кликом (графическая версия)

## 🎤 Использование

### CLI версия (консоль)
1. Дождитесь: "Ассистент готов к работе" 🔊
2. Скажите: **"JARVIS, [ваш вопрос]"**
3. Примеры:
   - "JARVIS, какое сейчас время?"
   - "JARVIS, расскажи про Python"
   - "JARVIS, открой Google"

### GUI версия (графический интерфейс)
1. Нажмите кнопку **"▶ Start Jarvis"**
2. Скажите: **"JARVIS, [ваш вопрос]"**
3. Смотрите логи в окне приложения

## 🔧 Параметры сборки

### Что включается в .exe:
✅ Все Python зависимости
✅ Модули ассистента (assistant/)
✅ Конфигурация (config.py)
✅ Модели TTS (Piper, Edge-TTS)
✅ Распознавание речи (SpeechRecognition)
✅ AI клиент (Groq API)

### Что НЕ включается:
❌ Файл `.env` с API ключом (нужно создать отдельно)
❌ Логи и кэш
❌ Виртуальное окружение

## 🐛 Решение проблем

### Ошибка: "ModuleNotFoundError"
```bash
# Переустановите зависимости
.venv\Scripts\python.exe -m pip install -r requirements.txt
# Пересоберите
pyinstaller JarvisCLI.spec --clean
```

### Ошибка: "API key not found"
Создайте файл `.env` рядом с `.exe` и добавьте:
```env
GROQ_API_KEY=gsk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### Антивирус блокирует .exe
Это **ложное срабатывание**. PyInstaller упаковывает Python скрипты, что иногда определяется как подозрительное.
- Добавьте в исключения антивируса
- Или используйте ручную установку из исходников

## 🎯 Советы по распространению

### Для личного использования:
- Просто запустите `build.bat` и используйте `.exe`

### Для публикации:
1. Соберите `.exe` файлы
2. Создайте ZIP архив:
   ```
   Jarvis/
   ├── JarvisGUI.exe
   ├── JarvisCLI.exe
   ├── .env.example    (шаблон конфигурации)
   └── README.txt      (инструкция)
   ```
3. Распакуйте и используйте
