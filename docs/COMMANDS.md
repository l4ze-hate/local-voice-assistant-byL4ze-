# 🎤 Команды голосового ассистента JarviX

## 📋 Как работают команды

### Основной принцип:
1. **Wake Word:** Скажите **"JARVIS"** для активации
2. **Команда:** После wake word скажите команду
3. **Ответ:** Ассистент выполнит команду и ответит голосом

### Пример:
```
Вы: "JARVIS, какое сейчас время?"
Ассистент: "Сейчас 14:30" 🔊
```

## ⚙️ Конфигурация команд

### Wake Word (слово активации)

По умолчанию: **"JARVIS"**

Можно изменить в `config.py`:
```python
WAKE_WORD = "JARVIS"  # Ваше слово активации
```

### Язык распознавания

По умолчанию: **Русский (ru-RU)**

Можно изменить в `config.py`:
```python
LANGUAGE = "ru-RU"  # Язык распознавания речи
```

## 🚀 Создание собственных команд

### Добавить свою встроенную команду

Откройте `assistant/commands.py` и добавьте:

```python
def execute(command):
    command = command.lower()

    # Ваш код здесь
    if "ваша команда" in command:
        # Действие
        return "Ваш ответ"

    # ... остальные команды
```

**Пример - открытие нового сайта:**
```python
if "открой википедию" in command:
    webbrowser.open("https://www.wikipedia.org")
    return "Открываю Wikipedia"
```

## 💡 Советы по использованию

### ✅ Для лучших результатов:

1. **Говорите чётко** - ассистент лучше распознавает
2. **Начните с "JARVIS"** - это активирует ассистент
3. **Подождите ответа** - после "Слушаю команду" говорите
4. **Используйте простые фразы** - легче распознать

### ❌ Чего избегать:

- ❌ Говорить слишком быстро
- ❌ Фоновый шум (музыка, TV)
- ❌ Сложные технические термины без контекста
- ❌ Несколько команд подряд без паузы

## 🔍 Решение проблем с командами

### ❌ Ассистент не распознаёт команду

**Решение:**
1. Проверьте микрофон: `python verify_setup.py`
2. Говорите чётче и медленнее
3. Попробуйте сформулировать иначе

### ❌ AI не отвечает

**Решение:**
1. Проверьте `.env` файл с API ключом
2. Убедитесь, что есть интернет
3. Проверьте лимит: 60 запросов/мин (Groq)

### ❌ Ассистент активируется сам

**Решение:**
1. Измените wake word в `config.py`
2. Убедитесь, что микрофон не слушает сам себя
3. Уменьшите громкость динамиков
