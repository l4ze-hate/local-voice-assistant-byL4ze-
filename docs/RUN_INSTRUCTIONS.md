# 🚀 Быстрый запуск

## Один универсальный батник

**Двойной щелчок по `run.bat`**

```
run.bat → Выбираешь режим (GUI или CLI) → Приложение запускается
```

---

## Выбор режима

При запуске батника увидишь меню:

```
Choose mode:
  [1] GUI - Graphical Interface (recommended)
  [2] CLI - Voice Console (direct commands)

Enter choice (1 or 2): _
```

### [1] GUI режим - Графический интерфейс (рекомендуется)

**Особенности:**
- ✅ Графический интерфейс
- ✅ Кнопки и текстовое поле
- ✅ Удобное управление
- ✅ Визуальный лог действий

**Как использовать:**
- Введи команду в текстовое поле
- Или скажи команду голосом
- Используй формат: "JARVIS, [вопрос]"

---

### [2] CLI режим - Голосовая консоль

**Особенности:**
- 🗣️ Чистая командная строка
- 🎙️ Голосовой интерфейс (только голос)
- ⚡ Минимальные ресурсы
- 📝 Полные логи

**Как использовать:**
1. Дождитесь сообщения "Ассистент готов к работе" 🔊
2. Скажите в микрофон: **"JARVIS, [вопрос]"** 🎤
3. Получите голосовой ответ 🔊

**Примеры команд:**
```
"JARVIS, какое сейчас время?"
"JARVIS, расскажи про Python"
"JARVIS, открой Google в браузере"
"JARVIS, помоги с кодом"
```

---

## ⚙️ Что делает батник?

При первом запуске батник автоматически:

1. **Создает виртуальное окружение** `.venv/`
2. **Устанавливает пакеты** из `requirements.txt`
3. **Устанавливает Piper TTS** (синтез речи)
4. **Проверяет конфигурацию** (`.env` файл)
5. **Запускает выбранное приложение**

---

## 🔧 Первый запуск

### Шаг 1: Запусти батник
```
Двойной щелчок → run.bat
```

### Шаг 2: Выбери режим
```
Enter choice (1 or 2): 1    [для GUI]
или
Enter choice (1 or 2): 2    [для CLI]
```

### Шаг 3: Первая установка
При первом запуске батник установит пакеты (~1-2 минуты)

### Шаг 4: Добави API ключ (если требуется)
Если батник попросит API ключ:
1. Перейди на https://console.groq.com/keys
2. Скопируй ключ (начинается с `gsk_`)
3. Вставь в файл `.env`
4. Перезапусти батник

## ✅ Если что-то не работает

### Нет звука?
- Проверь громкость Windows
- Проверь колонки подключены
- Попробуй режим GUI вместо CLI

### Микрофон не работает?
- Проверь микрофон в Windows Settings → Sound
- Дай приложению разрешение на использование микрофона
- Используй USB микрофон для лучшего качества

### Ошибка "Invalid API key"?
- Проверь файл `.env`
- Убедись ключ начинается с `gsk_`
- Получи новый ключ на https://console.groq.com/keys
- Проверь есть интернет

### Первый запуск зависает?
- Это нормально! Установка пакетов может занять 2-3 минуты
- Дождись завершения

## 💡 Советы

1. **Создайте ярлык** на рабочем столе
   - Правый клик на `run.bat`
   - "Отправить" → "Рабочий стол (ярлык)"

2. **Добавьте в автозагрузку**
   - Скопируй `run.bat` в папку автозагрузки
   - `%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup`

3. **Используйте USB микрофон**
   - Preferably Fifine USB microphone
   - Лучше качество распознавания речи
