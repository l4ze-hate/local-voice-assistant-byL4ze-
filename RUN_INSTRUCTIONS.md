# 🚀 Быстрый запуск через батники

## Два способа запуска

### 1️⃣ GUI версия (рекомендуется) - `run_app.bat`

**Двойной щелчок по `run_app.bat`**

```
run_app.bat → Автоматическая установка → GUI интерфейс
```

**Особенности:**
- ✅ Графический интерфейс
- ✅ Удобное управление
- ✅ Визуальный лог действий
- ✅ Эстетичный дизайн

**Процесс:**
1. Батник создаст виртуальное окружение (если нет)
2. Установит все пакеты (включая Piper TTS)
3. Проверит конфигурацию
4. Запустит GUI приложение

---

### 2️⃣ CLI версия (командная строка) - `run_cli.bat`

**Двойной щелчок по `run_cli.bat`**

```
run_cli.bat → Автоматическая установка → Голосовая консоль
```

**Особенности:**
- 🗣️ Чистая командная строка
- 🎙️ Голосовой интерфейс
- ⚡ Минимальные ресурсы
- 📝 Полные логи

**Как использовать CLI:**
1. Дождитесь сообщения "Ассистент готов к работе"
2. Скажите в микрофон: **"JARVIS, [вопрос]"**
3. Получите голосовой ответ

**Примеры команд:**
```
🎤 "JARVIS, какое сейчас время?"
🎤 "JARVIS, расскажи про Python"
🎤 "JARVIS, открой Google в браузере"
🎤 "JARVIS, помоги с кодом"
```

---

## ⚙️ Что делают батники?

Оба батника автоматически:

1. **Создают виртуальное окружение** `.venv/` (если его нет)
2. **Устанавливают пакеты** из `requirements.txt`
3. **Устанавливают Piper TTS** (для синтеза речи)
4. **Проверяют конфигурацию** (`.env` файл)
5. **Запускают приложение**

---

## 🔧 Первый запуск

### Если вы запустили батник в первый раз:

**Может потребоваться:**
1. ⏳ Установка пакетов (может занять 1-2 минуты)
2. 🔑 Добавить Groq API ключ в `.env` файл

**Где взяти Groq API ключ:**

1. Перейдите на https://console.groq.com/keys
2. Войдите в аккаунт (или создайте)
3. Скопируйте API ключ (начинается с `gsk_`)
4. Вставьте в файл `.env`:

```env
GROQ_API_KEY=gsk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

5. Сохраните и снова запустите батник

## ✅ Если что-то не работает

### Нет звука?
- Проверьте громкость Windows
- Проверьте колонки подключены

### Микрофон не работает?
- Проверьте микрофон в Windows Settings → Sound
- Убедитесь, что дали приложению разрешение

### Ошибка "Invalid API key"?
- Проверьте `.env` файл
- Убедитесь, что ключ начинается с `gsk_`
- Получите новый ключ на https://console.groq.com/keys

### Зависает при загрузке?
- Это нормально! Первый запуск устанавливает пакеты
- Может занять 2-3 минуты
