## 🚀 КАК ЗАПУСТИТЬ

### Вариант 1: Командная строка (CLI)

```bash
python main.py
```

**Что происходит:**
1. Услышите: "Ассистент готов к работе" 🔊
2. Программа слушает микрофон
3. Скажите: "JARVIS, [ваш вопрос]"
4. Ассистент ответит голосом

**Примеры команд:**
- "JARVIS, какое сейчас время?"
- "JARVIS, расскажи про Python"
- "JARVIS, открой Google"
- "JARVIS, какая сегодня погода?"

### Вариант 2: Графический интерфейс (GUI)

```bash
python app_gui.py
```

Или используйте батник:
```bash
run_app.bat
```

## 🎤 ВАЖНО!

### Требования к произношению:

1. **Говорите четко** - система распознает речь через Google Speech API
2. **Начните с "JARVIS"** - это wake word (команда активации)
3. **Говорите по-русски** - язык установлен на русский (ru-RU)
4. **Говорите если система слушает** - дождитесь "Слушаю команду"

### Пример правильной команды:

```
[Вы слышите] 🔊 Ассистент готов к работе
[Вы говорите] 🎤 "JARVIS, помоги с кодом на Python"
[Система слышит] 🔊 "Распознал: JARVIS, помоги с кодом на Python"
[Система говорит] 🔊 "Слушаю команду"
[Система отвечает] 🔊 "Вот пример кода на Python..."
```

## 🔊 ГОЛОСОВЫЕ УВЕДОМЛЕНИЯ

Ассистент произносит на каждом этапе:

| Сообщение | Когда |
|-----------|-------|
| "Ассистент готов к работе" | При старте |
| "Распознал: [текст]" | После распознавания голоса |
| "Слушаю команду" | После обнаружения "JARVIS" |
| "Выполняю: [действие]" | Когда обезжает команду |
| "Ищу ответ в базе знаний" | При запросе к AI |
| "[ОТВЕТ НА ВОПРОС]" | Голосовой ответ |
| "До встречи!" | При выходе |

## ✅ ВСТРОЕННЫЕ КОМАНДЫ

Без "JARVIS" в начале (быстрые команды):

- **"время"** → Произносит текущее время
- **"дата"** → Произносит текущую дату
- **"выход"** или **"пока"** → Завершить программу

Всё остальное обрабатывается через AI (нужен "JARVIS" в начале):
- **"JARVIS, [вопрос]"** → Запрос к Groq API

## 📝 ТЕХНИЧЕСКИЕ ДЕТАЛИ

**Компоненты:**
- **AI**: Groq API (llama-3.3-70b-versatile) - БЕСПЛАТНО
- **TTS**: Piper TTS (offline) + Edge-TTS (backup)
- **STT**: Google Speech API + sounddevice
- **Язык**: Русский (ru-RU)
- **Wake word**: "JARVIS"

**Производительность:**
- Старт: ~2-3 сек
- Распознавание: ~1-2 сек
- AI ответ: 1-5 сек
- Синтез голоса: ~1-2 сек

**Полный цикл:** 5-12 секунд от вопроса до голосового ответа
