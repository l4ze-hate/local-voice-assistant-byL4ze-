# 📦 БАТНИКИ - ГОТОВО К ЗАПУСКУ

## ✅ Что добавлено

Два полностью автоматизированных батника для запуска голосового ассистента:

### 1. `run_app.bat` - GUI версия (рекомендуется)
- Графический интерфейс
- Удобное управление
- Комфортное использование

**Просто двойной щелчок по `run_app.bat`** ↓

### 2. `run_cli.bat` - CLI версия (голос)
- Чистая командная строка
- Голосовой интерфейс
- Минимум ресурсов

**Просто двойной щелчок по `run_cli.bat`** ↓

---

## 🤖 Что делают батники?

Оба батника **автоматически:**

✅ Создают виртуальное окружение (если его нет)
✅ Устанавливают Python пакеты
✅ Устанавливают Piper TTS (синтез речи)
✅ Проверят конфигурацию
✅ Запускают приложение

---

## 📝 Первый запуск

### Шаг 1: Двойной щелчок по батнику
```
run_app.bat  ← GUI (проще начать отсюда)
```

Или:
```
run_cli.bat  ← CLI (для опытных)
```

### Шаг 2: Дождитесь установки
Первый запуск установит все пакеты (~1-2 минуты)

### Шаг 3: Добавьте API ключ (если требуется)
Если батник запросит API ключ:
1. Перейдите на https://console.groq.com/keys
2. Получите бесплатный ключ
3. Вставьте в файл `.env`
4. Перезапустите батник

### Шаг 4: Начните использовать!
Для CLI:
```
Скажите: "JARVIS, какое сейчас время?"
```

Для GUI:
- Используйте кнопки и текстовое поле

---

## 📂 Что создается автоматически

При первом запуске батник создает:

```
.venv/              ← Виртуальное окружение с Python
.env                ← Конфигурация (если не существует)
.logs/              ← Логи приложения
.data/              ← Кэш ответов AI
.cache/piper_voices/← Модели Piper TTS
```

## 🎯 Рекомендации

| Пользователь | Батник | Причина |
|--------------|--------|---------|
| Новичок | `run_app.bat` | GUI проще использовать |
| Опытный | `run_cli.bat` | Голосовая консоль быстрее |

## 💡 Советы

1. **Создайте ярлык** на рабочем столе:
   - Щелкните правой кнопкой на батнике
   - "Отправить" → "Рабочий стол (создать ярлык)"

2. **Добавьте в автозагрузку:**
   - Скопируйте батник в `%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup`

3. **Используйте USB микрофон** для лучшего качества
